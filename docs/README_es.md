
### Comando evaluaci√≥n: `eval <env-id> [opciones]`

Eval√∫a un entrenamiento existente cargando la Q-Table/pesos desde el directorio de `run` correspondiente. Por defecto, se abre la ventana (modo `human`) y se visualiza al agente usando sus pesos.

- **Uso b√°sico**: `eval <env-id> [opciones]`
